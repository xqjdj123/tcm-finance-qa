# 中药企业财务智能问答系统 🏭📊

> 基于 LLM Agent + RAG 的中药上市公司财务分析智能问答系统  
> 开发周期：2026年4月 — 2026年5月（约2个月）

---

## 📋 项目简介

本项目旨在构建一个**自然语言驱动的中药企业财务分析助手**。用户可以用中文提问，系统自动理解意图、检索研报、查询数据库并返回结构化答案（含图表）。

**典型问答示例：**
- *"2024年营收排名前五的中药企业是谁？"*
- *"片仔癀过去三年的净利润趋势如何？"*
- *"同仁堂的毛利率是多少？有没有同行对比？"*

## 🧩 模块详解

### 1️⃣ Agent 推理引擎 `agent/`
- 基于 ReAct 推理模式，支持多步工具调用
- 内置 **意图识别**（查询/对比/趋势/排行）
- 跨模块协作：NER → Schema Linking → SQL 生成 → 结果格式化

### 2️⃣ RAG 检索增强 `rag_pipeline/`
- **双路检索**：FAISS（稠密向量） + BM25（稀疏关键词）
- 向量模型：Ollama bge-m3（本地部署，无需HuggingFace）
- 研报预处理：页脚截断、语义分块、别名映射
- 产出：4306 向量 + 4306 BM25 文档

### 3️⃣ PDF 提取 `pdf_extractor/`
- 适配两种财报格式：**上交所**（表格内嵌） vs **深交所**（纯文本）
- 基于规则 + LLM 回退的字段映射
- 提取字段：资产、负债、营收、净利润、现金流等 50+ 指标

### 4️⃣ NER 命名实体识别 `models/ner_model/`
- 支持三类实体：公司名（含别名）、年份、财务指标
- 别名表覆盖：片仔癀、云南白药、同仁堂、白云山等 18 家中药上市公司
- 修复了 OCR 识别错误：片仑瘦→片仔癀、九苙堂→九芝堂

### 5️⃣ 前端界面 `static/ + templates/`
- 简洁的 Web 聊天界面
- 支持 markdown 渲染 + 图表嵌入

## 📓 开发札记

### Day 1-3：脚手架搭建
- 决定技术栈：Flask + Agent 架构
- 设计数据库 Schema：资产负债表、利润表、现金流量表、核心指标表
- **遇到的坑**：原始 Excel 表中字段命名不统一（中文/英文/缩写混用），花了一天梳理字段映射

### Day 4-6：PDF 财报提取
- 调试上交所财报 PDF：表格提取后行列错位，增加了 `pdf_segmenter.py` 做段落级解析
- **经典 Bug**：`extract_sse.py` 和 `extract_szse.py` 需要两套完全不同逻辑，一度想放弃统一方案
- 最终方案：先用规则提取，再用 LLM fallback 纠正异常值

### Day 7-10：RAG 研究与落地
- 初期尝试直接全文检索，效果很差
- 改用 FAISS + BM25 **双路召回**：BM25保精准率，FAISS 保召回率
- Chunk size 调优：从 1024→512→256，最终锁定 384（语义最完整）
- 页脚截断是个大坑——研报每页底部都有"本公司具备……"，匹配到就截断。后来改成**按行扫描 + 段落边界截断 + 10%长度守卫**

### Day 11-14：Agent 与 NL2SQL
- 意图识别：最初只支持简单问答，后来迭代支持对比、趋势、排行
- Schema Linking：让 LLM 理解"毛利率"对应哪个字段哪个表
- **记忆深刻的 Bug**：生成的 SQL 不带 WHERE 条件，查出全表数据。后来加入 Schema 过滤 + 字段验证
- 别名问题：研报里"民族瑰宝"=片仔癀、"云药"=云南白药，建了 50+ 条别名映射

### Day 15-18：测试与打磨
- 编写了 100+ 条回归测试用例（`test_all_questions.py`）
- 针对每个模块做单元测试：NER 准确率、SQL 正确率、RAG 召回率
- 数值格式化：解决了"一亿八千万"、"同比增长18.5%"等中文数字解析
- **最终指标**：NER F1 > 0.92，SQL 准确率 > 85%，端到端问答准确率 > 80%

### 心得体会
> *"这个项目最大的收获不是技术，而是**如何把一个模糊的需求拆解成可执行的模块**。一开始'做个财务问答系统'听起来很简单，真做起来才发现：PDF格式差异、公司名别名、研报页脚污染……每一个小问题都足够折腾一整天。"*
>
> *"迭代是王道——不要追求一次性完美，先跑通再优化。第一个版本只能回答完全匹配的问题，但**先跑通给信心**，后面 RAG、NER、Schema Linking 才一个一个加上去。"*
